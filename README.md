# 助教排班系统

一个用于管理助教名单、时间段和排班安排的网页工具。

## 功能

- 手动添加、删除助教
- 上传 Excel/CSV 批量导入助教名单
- 点击助教姓名安排或取消某个时间段
- 自定义时间段名称和时间
- 统计总助教、已安排、空闲人数
- 导出排班文本
- 服务端共享保存
- 多人打开同一页面时实时同步

## 本地运行

需要 Node.js 18 或更新版本。

```bash
npm start
```

启动后打开：

```text
http://localhost:3000
```

## 数据保存

共享排班数据默认保存在：

```text
data/schedule.json
```

`data/` 已加入 `.gitignore`，不会被提交到仓库。部署到服务器时，请确保这个目录可以写入。

## 直接打开 HTML

如果直接双击打开 `index.html`，页面仍然可以使用，但只能保存到当前浏览器本地，无法多人共享。

要多人共同维护同一份排班，请使用 `npm start` 或部署 `server.js`。

## 部署提示

服务器需要能运行 Node.js。常见部署方式：

1. 上传整个项目到服务器
2. 运行 `npm start`
3. 用 Nginx 或服务器面板把域名反向代理到 `http://127.0.0.1:3000`

如果服务器需要直接监听外网地址，可以这样启动：

```bash
HOST=0.0.0.0 PORT=3000 npm start
```

当前版本没有账号权限控制。如果页面公开访问，任何能打开页面的人都可以修改排班。
